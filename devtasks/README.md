# 🚀 DevTasks — Secure Command Runner

Interface web para execução segura de comandos com backend protegido.

## ✨ Features

- 🔐 Token de autenticação
- 🧠 Whitelist de comandos
- ⏱️ Timeout de execução
- 📊 Monitor de performance (CPU/RAM)
- 📝 Histórico exportável
- 🛑 Controle de execução
- 🎨 UI Matrix Black Neon

## 🧱 Stack

- Node.js  
- Express  
- Vite  
- Vanilla JS  
- CSS Neon  

## ▶️ Como rodar local

### Backend

```bash
cd devtasks/backend
npm install
node server.mjs
